# CMPSC470_FinalProject
## "Make Math Easy" (MME) Programming Language

 - A Python-based, natural-language-like programming language that allows for simple algebra/calculus problem solving
 - Follows the following workflow of an interpreter (Input --> Lexical Analysis -> Syntax Analysis -> Translation --> Ouput)
 - Simulates the process of an interpreter using Python code to read a .txt file as an MME source code file line-by-line
 - Uses: engineering/science fields for efficient and accurate mathematical calculations, useful for individuals without prior programming knowledge (easy-of-use syntax and rules), mathematics teachers/professors for grading
 - Future Developments: more types of problems to solve (higher complexity), further step-by-step analysis of how problems are solved rather than simply providing answers, etc.
